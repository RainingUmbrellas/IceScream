# IceScream

IceScream is a Halloween themed ice cream shop.
This shop boasts 5 flavors.

Ice cream flavors

1. Graveyard Chocolate Crunch
2. Witch's Berry Brew
3. Ghosty Vanilla
4. Mummy's Cookies & Cream
5. Dracula's Kiss

You can get 1 or 2 scoops.
You have the option to get your ice cream in a waffle cone or cup.

Users should be able to create a Dine-in, takeaway, pick-up, or delivery order.
